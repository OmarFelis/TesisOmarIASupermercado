INSTRUCCIONES PARA AGREGAR LOGO:

1. Guarda el logo de tu universidad como: logo_universidad.png
2. Colócalo en esta carpeta (assets/)
3. Formatos soportados: PNG, JPG, JPEG
4. Tamaño recomendado: cualquier tamaño (se redimensiona automáticamente a 100x100)
5. Para mejor resultado usa PNG con fondo transparente

El logo aparecerá en la esquina superior derecha de la aplicación.

Si no hay logo, la aplicación funcionará normalmente sin mostrar nada.
